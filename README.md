# cpp-multiple-classes-project
A simple C++ Object-Oriented Programming project demonstrating multiple classes (Student, Book, and Laptop), object creation, storing data in objects, and displaying information using cout.

# Multiple Classes Project in C++

## Description
This is a simple C++ OOP project that demonstrates:

- Creating multiple classes
- Declaring data members
- Creating objects
- Storing values in objects
- Displaying information using cout

## Classes Used

### Student
- name
- age

### Book
- title
- price

### Laptop
- brand
- ram

## Output
Name : sadiya
Age : 18
Title : c++
Price : 50
Brand : HP
RAM : 16GB
